# Bingo Generator


This is a simple Bingo card generator written in Python. It creates a standard 5x5 Bingo card with random numbers in each column according to traditional Bingo rules.

It also feature a static HTML page to pick and display a random number between 1 and 76 for Bingo games.

## Features

- Generates a 5x5 Bingo card with appropriate number ranges for each column.
- The center space is a free space.
- Outputs the Bingo card in a pdf format.


- Includes a static HTML page to pick a random number for Bingo games.
- Verification of drawn numbers to avoid duplicates.
- Verification of drawn numbers against the Bingo card.
- Simple and user-friendly interface.
- Responsive design for mobile and desktop use.
